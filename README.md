# AWP-Instakill
<h2><b>Descrição:</b></h2>
O plugin de AWP Instakill faz com que a arma AWP dê 1000 de dano, ou seja se alguém acertar num inimigo furado ou na perna, essa pessoa irá morrer na mesma.
